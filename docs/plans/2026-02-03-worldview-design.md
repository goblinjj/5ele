# 无极 - 世界观设定

## 开场（游戏开始时显示）

无极生太极，太极生两仪，两仪生四象，四象生五行，五行化万物。

这是世界的法则，亘古不变。

然而五行失衡，万物病变。世界正在崩坏。

而你——本源的意志残魂——被赋予了使命：

**夺回五行之力，让世界归于平衡。**

---

## 结语（游戏结束时显示）

使命完成。你将回归本源，不留下任何痕迹。

你不知道自己从何而来。
你从未见过本源的模样。
你只是被赋予了使命，然后去做了。

新的残魂会诞生，背负同样的使命，踏上同样的路。

---

## 第一章：五行天地

太初之时，世界由五行之力交织而成——金、木、水、火、土相生相克，维系着万物的平衡与运转。五行流转不息，世界便生生不灭。

然而不知何时，五行之力开始**失衡**。失衡的能量逐渐变异，产生了自我意志，化为形形色色的**妖异**。这些妖异并非外来入侵者，而是世界自身的**病变**——它们渴望吞噬更多的五行之力来壮大自身，甚至觊觎世界本源的意志。

随着妖异蔓延，世界日渐衰弱，濒临崩坏。

---

## 第二章：本源之子

世界濒临崩坏之际，本源意志开始自救——它将自身分裂成无数缕**意志残魂**，散落于天地之间。

每一缕残魂都是世界的一部分，天生拥有操控五行之力的能力。它们的使命只有一个：**夺回被妖异侵占的五行之力，归还于世界。**

但残魂并非没有自我。每一缕残魂在诞生时，都会产生自己的**意念与想法**。它们知道，谁夺回的五行之力越多，谁的意志就越强——当使命完成、回归本源时，**最强的那缕意志将引导世界重生的方向**。

因此，残魂之间既是同源的手足，也是彼此竞争的对手。

---

## 第三章：双向猎杀

妖异是变异的五行之力所化，而意志残魂是世界本源的碎片——两者都渴望吞噬对方。

妖异想要夺取残魂，将本源意志据为己有，彻底掌控这片天地。残魂则要击溃妖异，磨除其变异的意志，将五行之力净化归元。

这是一场**双向的猎杀**，没有旁观者。

当妖异被击败，变异的意志消散，残留的五行之力便恢复纯粹。残魂可以将这些能量塑形为**器物**——武器、铠甲、法宝——环绕自身，以战养战。

残魂亦可将器物中的五行之力**重组**或**归元**：重组可凝练出更强的器物，归元则将能量化为己用。但五行能量本就不稳定，重组时常有失败之险。

---

## 第四章：广袤与轮回

这个世界远比想象中广袤。无数的**气脉区域**遍布天地，每一处都有妖异盘踞、病变丛生。

一缕残魂穷尽一生，也只能净化一小片区域。当残魂击败区域中最强大的**病变核心**，便完成了使命，其意志将**回归本源**，与世界融为一体。

但这不是终结。

世界的其他角落，新的病变仍在滋生，新的妖异仍在壮大。本源会诞生新的残魂，前往新的区域，继续这场永无止境的净化之旅。

每一缕残魂的牺牲都有意义——它们夺回的五行之力，会让世界离真正的平衡更近一步。只是那一天何时到来，无人知晓。

---

## 第五章：命脉交汇

当多缕残魂同处一片区域时，每当一轮战斗结束，世界的**五行脉络**会短暂显现——这是世界残存的"呼吸"。

在脉络显现的刹那：

- **共鸣** — 所有残魂被脉络的光芒照亮，彼此能感知到对方的气息与形态。这并非主动窥探，而是同源意志之间的被动共鸣。

- **抉择** — 脉络显示出多条通往不同**气穴**的路径，每一处气穴都蕴含着不同的机缘。残魂们各自选择想要前往的方向，但在脉络稳定之前，谁也不知道其他残魂的选择。

- **裁决** — 若多缕残魂选择了同一条脉络，世界本身会进行裁决——以残魂与该脉络的**契合度**决定谁能通行。落选者会被脉络的反冲力推向其他空缺的气穴。

世界并非偏袒谁。它只是本能地运转，等待最终脱颖而出的那缕意志。

---

## 第六章：殊途同归

每片区域的深处，都盘踞着一个最为庞大的**病变核心**——它吸纳了最多的变异五行之力，是这片区域妖异的源头。

净化病变核心，是每一缕残魂的最终目标。

当多缕残魂同处一片区域时，它们注定会在清剿沿途妖异后，**同时盯上这同一个目标**。于是，残魂与病变核心的决战，也就不可避免地成为了残魂之间的混战。

没有谁刻意安排这一切。只是多缕意志，殊途同归。

在这场乱战中，残魂既要击溃病变核心，也要与同源的对手争夺五行之力。因为它们都清楚——谁夺取的越多，谁的意志就越强，回归本源时就越有资格主导世界的未来。

---

## 术语表

| 术语 | 解释 |
|------|------|
| **无极** | 世界的名称。道家指混沌未分的本源状态，也暗示无穷无尽、永无止境 |
| **五行** | 金、木、水、火、土。构成世界的基础能量，相生相克 |
| **本源** | 世界的意志核心。无人见过其真正的模样 |
| **意志残魂（残魂）** | 本源分裂出的意志碎片，即玩家。被赋予夺回五行之力的使命 |
| **妖异** | 五行之力变异后产生意志所化的怪物，是世界的病变 |
| **病变核心** | 区域中最强大的妖异，吸纳了最多变异的五行之力（即Boss） |
| **气穴** | 五行脉络上的节点，蕴含不同的机缘（即地图节点） |
| **命脉/五行脉络** | 世界残存的能量通道，每轮结束时短暂显现 |
| **命脉交汇** | 脉络显现时，残魂们能感知彼此、选择去向的时刻 |
| **裁决** | 多个残魂选择同一气穴时，世界以契合度决定归属 |
| **归元** | 将器物中的五行之力打散吸收（即吞噬） |
| **重铸** | 将多件器物的五行之力融合凝练（即合成） |
| **回归** | 残魂完成使命后，意志融入本源（即游戏结束） |

---

## 设计说明

### 核心隐喻

「无极」的世界观映射现实：人们为他人设定的目标奋斗一生，最终什么都带不走。新的生命诞生，重复同样的轨迹。这一切有意义吗？也许没有，但我们依然在做。

### 游戏机制对应

| 游戏机制 | 世界观解释 |
|----------|------------|
| 肉鸽无存档 | 残魂完成使命后回归本源，不留痕迹 |
| 重复游玩 | 世界广袤，需要无数残魂净化不同区域 |
| 多人竞争 | 残魂各有意志，争夺五行之力以主导世界未来 |
| 击败怪物 | 磨除变异意志，净化五行之力 |
| 装备掉落 | 纯粹的五行之力被残魂塑形为器物 |
| 合成系统 | 五行能量重组，不稳定故有失败率 |
| 吞噬系统 | 将器物归元，能量化为己用 |
| 选点机制 | 命脉交汇时选择气穴 |
| 选点冲突 | 世界裁决契合度 |
| Boss战 | 净化病变核心 |
| 多人混战 | 殊途同归，同时盯上同一目标 |
