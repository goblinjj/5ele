## 素材规格建议

### 怪物素材
- 尺寸: 128x128 像素 (或 256x256 高清版)
- 格式: PNG (透明背景)
- 风格: 像素风格或手绘卡通风格
- 需要: 待机帧、攻击帧、受伤帧 (可选)

### 装备图标
- 尺寸: 64x64 像素
- 格式: PNG (透明背景)
- 风格: 与怪物素材风格统一

### 五行特效
- 尺寸: 可变 (建议 64x64 基础)
- 格式: PNG 序列帧或 GIF
- 内容: 金光、木叶、水波、火焰、土石
