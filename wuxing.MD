# 五行被动技能列表

## 🟡 金属性 (Metal)


| 技能   | 解锁等级 | 效果说明                                                                                  |
| :----- | :------- | :---------------------------------------------------------------------------------------- |
| `破金` | **Lv1+** | 无视敌人`10%/20%/35%/50%/70%` 防御                                                        |
| `裂伤` | **Lv3+** | `15%/25%/100%` 概率造成流血，每回合造成 `5%/5%/10%` 最大生命伤害，可叠 5 层。Lv5 伤害翻倍 |

---

## 🟢 木属性 (Wood)


| 技能   | 解锁等级 | 效果说明                                                            |
| :----- | :------- | :------------------------------------------------------------------ |
| `生机` | **Lv1+** | 每回合恢复`3%/5%/8%/12%/15%` 最大生命（最少1点）。Lv4+ 低血量时翻倍 |
| `不朽` | **Lv3+** | 每回合首次受到致命伤害时，保留 1 点生命值                           |
| `涅槃` | **Lv5**  | 战斗中死亡时，以`50%` 生命值复活（每场战斗一次）                    |

---

## 🔵 水属性 (Water)


| 技能   | 解锁等级 | 效果说明                                                                 |
| :----- | :------- | :----------------------------------------------------------------------- |
| `寒息` | **Lv1+** | `15%/25%/35%` 概率减速 `20%/30%`，持续 2 回合。Lv3+ 对减速目标 +15% 伤害 |
| `冰封` | **Lv4+** | `20%/30%` 概率冻结敌人（跳过回合）                                       |
| `冰碎` | **Lv5**  | 冻结解除时造成`100%` 攻击力额外伤害                                      |

---

## 🔴 火属性 (Fire)


| 技能   | 解锁等级 | 效果说明                                                             |
| :----- | :------- | :------------------------------------------------------------------- |
| `灼烧` | **Lv1+** | 攻击施加灼烧，每回合造成`3%/5%/5%/5%/8%` 最大生命伤害，持续 2/3 回合 |
| `烈焰` | **Lv3+** | 灼烧可叠加至 3 层                                                    |
| `引爆` | **Lv4+** | 3 层时引爆造成`20%/35%` 最大生命伤害                                 |
| `余烬` | **Lv5**  | 引爆后敌人进入余烬状态，火伤害 +50%                                  |

---

| 技能   | 解锁等级 | 效果说明                       |
| :----- | :------- | :----------------------------- |
| `坚韧` | **Lv1+** | 减伤`10%/18%/25%/35%/45%`      |

| `不动` | **Lv4+** | HP > 70% 时免疫控制            |
| `蓄力` | **Lv5**  | 受击 3 次后下次攻击 +100% 伤害 |
